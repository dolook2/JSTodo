# JSTodo
To Do List 애플리케이션

## 기능
- 이름넣기  
- ToDoList 작성
- 현재 시간
- 기온 및 위치 불러오기

## 사용 기술
- Front-End : Vanila JavaScript
- Back-End : 없음
- Database : 없음
- Deployed : github.io

## Start
1. 레포지토리의 소스코드 pull
2. npm i
4. npm start
5. localhost:8080에서 확인
